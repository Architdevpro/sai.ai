# SAI Chat Assistant

Welcome to **SAI Chat Assistant**!  
This project is a modern, web-based AI assistant interface built using HTML and styled with Tailwind CSS.

## Features

- Responsive design with Tailwind CSS
- Animated chat interface (typing indicators, message transitions)
- Navigation bar with Home, About, and Features pages
- Easy to extend and customize

## Getting Started

1. **Clone the repository**:
    ```bash
    git clone https://github.com/Architdevpro/sai.ai.git
    cd sai.ai
    ```
2. **Open `index.html`** in your browser to launch the SAI Chat Assistant interface.

## Project Structure

- `index.html`: Main landing/chat interface
- `home.html`: Home page
- `about.html`: Information about the assistant
- `features.html`: List of capabilities and features

## Customization

You can easily modify styles and scripts in `index.html` and related files. Tailwind CSS is used via CDN for fast prototyping.
